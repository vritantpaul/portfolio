Projects

Cyclistic bikeshare case study

Cyclistic is a company which offers rental bike services in Chicago. Here I did a descriptive customer analytics to maximize cyclistics' membership distribution over casual riders and came up with insights and recommendations necessary to make that convert possible. The whole analysis is done in Python programming language.

Bellabeat fitness tracker case study

Bellabeat is a US based firm of wearable computers for women. Here I did an exploratory analysis on a dataset containing data from 30 participants, having recorded data of their heart rate, steps taken per day, BMI, sleep duration etc. for a span of 1 month. The whole analysis is performed using R programming language.

Netflix Titles Analysis

Netflix tops as the number one streaming service provider in today's world. I did an investigative analysis on 8K+ Netflix Titles both movies and TV shows. The analysis is summarized through Tableau Dashboarding. Spoiler alert: TV shows are the new thing outperforming movies, significant uprising ever since the inception of COVID and lockdown.

Amazon Bestselling Books Webscraping

Here I wrote a python script to scrape the data of Top 100 bestselling books of all time in Amazon. The scraping is done through 'Beautifulsoup' library. The scraper is set to scrape every 15 min. and will compile the data into a CSV file. Fields include: title, author, type, ratings, number of people rated & price.

Covid-19: Data Exploration

For this project I did a data exploration on COVID 19 data particularly death toll and vaccination rate across countries. The dataset is provided by Our World in Data. I used SQL queries to answer some questions and visualized the major findings using Tableau.
