Welcome

Hello! I’m Vritant, a storyteller fluent in data. My journey began in business administration, but during my master's, I discovered it wasn't where my passion lay. Thus after a lot of late night contemplations and a bit of existential crisis, I decided to take a different path and pursue a career in data science. I settled on this discipline as I felt like I have both the inclination and aptitude required to excel in this field. As the cliché goes, "Data is the new oil", I understand this first-hand from my reading of classics like Factfulness, and Enlightenment Now. Going forward I wish to hone my data skills, and work on intellectually stimulating and challenging problems with colleagues who share my enthusiasm for data. If our paths align, I look forward to working with you, and to the engaging conversations along the way.

Projects

Cyclistic Bikeshare Case Study

Cyclistic is a bike-share service in Chicago. I analyzed ride data to understand user behavior and recommended strategies to convert casual riders into annual members. The entire analysis was done in Python using tools like Pandas and Matplotlib.

Bellabeat Fitness Tracker Case Study

Bellabeat designs wellness devices for women. I explored activity and biometric data from 30 users to identify health trends and usage patterns. This R-based analysis offered insights to improve user engagement.

Ed-Tech Customer Analytics

This project focused on analyzing user interaction data from an ed-tech platform. I used Tableau to visualize KPIs and identify key touchpoints in the user journey, helping highlight areas to improve engagement.

Netflix Titles Analysis

Using data from over 8,000 Netflix titles, I explored trends in genre, release years, and content types. The interactive Tableau dashboard revealed how Netflix's global catalog has evolved over time.

YouTube to Notion Notes

This Python program summarizes YouTube videos with subtitles and sends the key points directly to Notion. It uses the YouTube Transcript API for fetching transcripts, ChatGPT for summarization, and the Notion API to format and store notes as bullet points.

Amazon Product Price Drop Notifier

A Python-based tool that scrapes Amazon product pages periodically and sends a Windows notification if the product price drops to or below a set threshold. The check is automated using Windows Task Scheduler.

COVID-19 Data Exploration

Using COVID-19 data from Our World in Data, I explored death tolls and vaccination rates across countries. I wrote SQL queries to extract insights and used Tableau to visualize key trends and comparisons.
